    Enum RotationFitMode

    Mode for determining final output size when rotating an image.

    Namespace: Microsoft.Psi.Imaging
    Assembly: Microsoft.Psi.Imaging.dll
    Syntax
    public enum RotationFitMode : int

    Fields

    Name Description
    Loose

    Output image will always be the maximum width/height regardless of the specified rotation amount. For instance, if rotating a 100x200 image by 10 degrees the output image size will be 224x224. That is the diagonal length and the amount of the rotation is irrelevant.
    Tight

    Output image will be fit exactly. Thus if we rotate a 100x200 image by 10 degrees the output image size will be 35x180.
