    Class ImageBase

    Represents an image, stored in unmanaged memory.

    Inheritance
    Object
    ImageBase

    Namespace: Microsoft.Psi.Imaging
    Assembly: Microsoft.Psi.Imaging.dll

    public abstract class ImageBase : IDisposable
    Remarks

    Using this class it is possible as to allocate a new image in unmanaged memory, as to just wrap provided pointer to unmanaged memory, where an image is stored.

    ImageBase(BitmapData, Boolean)

    Initializes a new instance of the ImageBase class.

    Declaration
    public ImageBase(BitmapData bitmapData, bool makeCopy = false)
    Parameters
    Type Name Description
    BitmapData bitmapData

    Locked bitmap data.
    Boolean makeCopy

    Indicates whether a copy is made (default is false).
    Remarks

    When the makeCopy parameter is false (default), the image simply wraps the bitmap data. As such, the bitmap data must stay locked for the duration of using the ImageBase object.

    If the makeCopy parameter is set to true, a copy of the bitmap data is made, and the bitmap data can be released right after the ImageBase has been constructed.

    ToBitmap(Boolean)

    Creates a from the image.

    Declaration
    public Bitmap ToBitmap(bool makeCopy = true)
    Parameters
    Type Name Description
    Boolean makeCopy

    Indicates whether to make a copy of the image data or not.
    Returns
    Type Description
    Bitmap

    A corresponding image.
    Remarks

    If the makeCopy parameter is set to true, then the method creates a copy of the image, so the stays valid even when the image gets disposed. However, setting this parameter to false creates a image which is just a wrapper around the image data. In this case, if the image is disposed, the will no longer be valid and accessing it will generate an exception.
