    Use the Rodrigues formula for transforming a given rotation from axis-angle representation to a 3x3 matrix. Where 'r' is a rotation vector: theta = norm(r) M = skew(r/theta) R = I + M * sin(theta) + M*M * (1-cos(theta)).
